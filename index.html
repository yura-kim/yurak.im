Projects

Motivate [Cordova, Ionic Framework, JavaScript, HTML, CSS]
January 2015 – Present

A diet and nutrition tracking Android application that sets itself apart from others by focusing on keeping the user motivated through progress tracking and a semi-anonymous messaging system. The team started with a basic idea for the application that led to a software requirements specification document using requirements elicitation techniques such as user interviews, persona creation, and talking to subject matter experts. High fidelity mock-ups were created following Android Material Design guidelines. Implementation is done using Cordova and the Ionic Framework.

Participation Tracker [Software Requirements and Architecture]
January 2015 – Present

A system that keeps track of the participation in class of students from grades K-6 through a web portal and Android tablet application which interface with a common database. The team created a software requirements specification document using requirements elicitation techniques such as user interview, persona creation, and research. The team is currently working to create an effective software architecture to model the system.

Fuzzer [Python]
October 2014

A web crawling fuzz testing tool designed to find weaknesses in a program's attack surface. The fuzzer checks for common exploits to vulnerability such as SQL injects, cross-site scripting, buffer overflows, and guessing input and authorization using a provided file of common words.
